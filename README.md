# Project Reading App

### Partie Back

- Pour aller dans environement

  ```
  cd back/
  ```

- Pour aller dans environement

  ```
  source .env/bin/activate
  ```

- Pour installer des libraries

  ```
  pip install -r requirements.txt
  ```

- Pour executer l'application

  ```
  export FLASK_APP=app
  export FLASK_ENV=development
  flask run
  ```

### Partie Front

- Pour aller dans environement

  ```
  cd front/
  ```

- Pour installer des libraries

  ```
  npm install
  ```

  ou

  ```
  yarn install
  ```

- Pour executer l'application

  ```
  npm run dev
  ```

  ou

  ```
  yarn dev
  ```
